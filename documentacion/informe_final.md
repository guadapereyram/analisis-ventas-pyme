# Informe Final – Análisis de Ventas en PyME

## Introducción

Este proyecto fue desarrollado con el objetivo de aplicar herramientas de gestión colaborativa, control de versiones y análisis de datos utilizando Python.

Se trabajó con GitHub para el versionado del código, Jira para la planificación y seguimiento de tareas, y Google Colab como entorno de ejecución.

## Objetivo

Analizar datos de ventas simulados de una PyME para obtener métricas útiles para la toma de decisiones.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Git
- GitHub
- Jira
- Google Colab
