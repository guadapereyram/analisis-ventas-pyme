# Análisis de Ventas en PyME

Proyecto académico orientado al análisis de ventas utilizando Python, GitHub, Jira y Google Colab.

## Objetivo
Analizar datos de ventas de una PyME para obtener información útil para la toma de decisiones.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub
- Jira
