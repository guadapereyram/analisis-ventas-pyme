# Análisis de Ventas en PyME

Proyecto académico orientado al análisis de datos de ventas utilizando herramientas de gestión colaborativa y control de versiones.

## Objetivo

Analizar información de ventas de una PyME para obtener métricas relevantes que permitan apoyar la toma de decisiones.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Git
- GitHub
- Jira
- Google Colab

## Funcionalidades implementadas

- Lectura de archivo CSV
- Procesamiento de datos con Pandas
- Cálculo de métricas:
  - Total de ventas
  - Promedio de ventas
  - Producto más vendido
  - Ventas por categoría
- Generación automática de gráfico
- Exportación de resultados

## Estructura del proyecto

```text
/scripts
/datos
/resultados
/documentacion
```

## Flujo de trabajo aplicado

- Planificación de tareas en Jira
- Uso de commits trazables vinculados a tareas
- Ejecución reproducible en Google Colab
- Versionado del proyecto mediante Git y GitHub
